# Netflix Homepage Clone

A responsive Netflix homepage clone built using HTML, CSS, and JavaScript. This project recreates the look and feel of Netflix's landing page and was created to practice front-end web development skills.

## Features

- Responsive design
- Modern Netflix-inspired user interface
- Navigation bar
- Hero section with call-to-action
- Interactive elements using JavaScript
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Purpose

This project was created as part of my web development learning journey to improve my understanding of:

- Semantic HTML
- CSS Flexbox and Grid
- Responsive Web Design
- DOM Manipulation
- Front-End Development Best Practices
